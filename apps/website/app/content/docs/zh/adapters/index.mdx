## 快速开始

### 安装

```bash tab="pnpm"
# 安装核心包和所需的适配器
pnpm add @amux/llm-bridge @amux/adapter-openai
```

```bash tab="npm"
npm install @amux/llm-bridge @amux/adapter-openai
```

```bash tab="yarn"
yarn add @amux/llm-bridge @amux/adapter-openai
```

### 使用

```typescript
import { createBridge } from '@amux/llm-bridge'
import { openaiAdapter } from '@amux/adapter-openai'
import { anthropicAdapter } from '@amux/adapter-anthropic'

// 创建桥接
const bridge = createBridge({
  inbound: openaiAdapter,
  outbound: anthropicAdapter,
  config: {
    apiKey: process.env.ANTHROPIC_API_KEY
  }
})

// 发送请求
const response = await bridge.chat({
  model: 'gpt-4',
  messages: [{ role: 'user', content: '你好！' }]
})
```

## OpenAI 兼容性

以下适配器基于 OpenAI 格式，可以无缝替换：

- **DeepSeek** - 完全兼容 OpenAI API
- **Moonshot** - 完全兼容 OpenAI API
- **Zhipu** - 完全兼容 OpenAI API
- **Qwen** - 部分兼容（有细微差异）
- **Gemini** - 部分兼容（需要格式转换）

<Callout type="warn">
虽然这些适配器声称兼容 OpenAI，但在某些细节上可能有差异（如系统消息处理、流式格式等）。建议查看各适配器的详细文档。
</Callout>

## 创建自定义适配器

如果您需要支持其他 LLM 提供商，可以创建自定义适配器：

```typescript
import type { LLMAdapter } from '@amux/llm-bridge'

export const myAdapter: LLMAdapter = {
  name: 'my-provider',
  version: '1.0.0',
  capabilities: {
    streaming: true,
    tools: true,
    vision: false,
    multimodal: false,
    systemPrompt: true,
    toolChoice: true
  },
  inbound: {
    parseRequest: (request) => { /* ... */ },
    parseResponse: (response) => { /* ... */ }
  },
  outbound: {
    buildRequest: (ir) => { /* ... */ },
    buildResponse: (ir) => { /* ... */ }
  },
  getInfo() {
    return {
      name: this.name,
      version: this.version,
      capabilities: this.capabilities
    }
  }
}
```

查看 [适配器 API 文档](/docs/zh/api/adapters) 了解更多详情。
