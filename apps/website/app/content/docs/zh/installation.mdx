## 前置要求

- Node.js 18 或更高版本
- pnpm、npm、yarn 或 bun

<Callout type="warn">
确保您已安装 Node.js 18+。您可以使用 `node --version` 检查版本。
</Callout>

## 安装核心包

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']}>
  <Tab value="pnpm">
    ```bash
    pnpm add @llm-bridge/core
    ```
  </Tab>

  <Tab value="npm">
    ```bash
    npm install @llm-bridge/core
    ```
  </Tab>

  <Tab value="yarn">
    ```bash
    yarn add @llm-bridge/core
    ```
  </Tab>

  <Tab value="bun">
    ```bash
    bun add @llm-bridge/core
    ```
  </Tab>
</Tabs>

## 验证安装

创建一个简单的测试文件来验证安装：

```typescript
import { createBridge } from '@llm-bridge/core'
import { openaiAdapter } from '@llm-bridge/adapter-openai'

console.log('✓ LLM Bridge 安装成功！')
console.log('✓ OpenAI 适配器：', openaiAdapter.name, 'v' + openaiAdapter.version)
console.log('✓ 能力：', openaiAdapter.capabilities)
```

运行它：

```bash tab="tsx"
tsx test.ts
```

```bash tab="node"
node --loader tsx test.ts
```

```bash tab="bun"
bun test.ts
```

您应该看到类似以下的输出：

```
✓ LLM Bridge 安装成功！
✓ OpenAI 适配器： openai v1.0.0
✓ 能力： { streaming: true, tools: true, ... }
```

## TypeScript 配置

<Callout type="info">
LLM Bridge 使用 TypeScript 编写，包含完整的类型定义。不需要额外的 `@types` 包。
</Callout>

确保您的 `tsconfig.json` 包含：

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext",
    "moduleResolution": "bundler",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true
  }
}
```
